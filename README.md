# 🇮🇷 Iranian Cities Population Map

## 📌 Project Overview

This project visualizes the population distribution of Iranian cities using geospatial data visualization techniques in Python.
It uses **Basemap** and **Matplotlib** to plot cities on a map and represent population intensity with color and size variation.

---

## 🎯 Objectives

* Load and process city-level dataset
* Analyze population statistics
* Visualize geographic distribution of cities
* Represent population using color intensity and marker size

---

## 📊 Features

* Geographical plotting using Basemap
* Scaled markers based on population
* Color mapping for population density
* City name annotations on map
* Basic statistical analysis of dataset

---

## 🧰 Libraries Used

* NumPy
* Pandas
* Matplotlib
* mpl_toolkits.basemap

---

## 🗺️ Visualization Output

The map shows:

* Location of Iranian cities (Longitude & Latitude)
* Population size encoded in marker size
* Population density encoded in color gradient

---

